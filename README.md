# JK Trade

Industrial trade house in Rajkot, Gujarat. Authorised dealer for **Dow Corning**, **Wacker**, **3M VHB**, **Tesla Tapes**, **Euro Bond ACP** and more. Since 2008.

This repository contains the public website.

## Stack

Pure static HTML/CSS/JS. No build step, no dependencies.

- `index.html` — single-page site
- `styles.css` — design system + 3D effects
- `script.js` — scroll progress, 3D card tilt, mobile nav, stat counters, form handoff
- `logo-mark.png` — circular brand emblem (used as favicon, nav, footer)
- `logo-wordmark.png` — full "JK TRADE" wordmark (reserved for future use)

## Run locally

Double-click `index.html` — opens in your default browser. That's it.
